🧮 원석T의 매쓰 바이브 (MATH VIBE)

"수학은 뭐다? EASY!" > 학생들의 수학 실력 향상을 위한 인터랙티브 웹 게임 프로젝트입니다.

📖 프로젝트 소개

MATH VIBE는 중/고등 수학 문제를 게임처럼 즐겁게 풀 수 있도록 설계된 웹 애플리케이션입니다.

단순한 문제 풀이를 넘어, 타임어택과 대결 요소를 도입하여 학생들의 흥미와 몰입도를 극대화했습니다.

🎮 주요 기능 (Game Modes)

1. 🏆 챌린지 모드 (Challenge Mode)

브론즈부터 챌린저까지 총 20단계의 레벨업 시스템.

난이도가 점진적으로 상승하며 한계를 시험하는 모드.

실시간 타임 게이지와 점수 시스템 적용.

2. 📚 유형 연습 (Practice Mode)

대수, 미적분, 기하, 확통 4대 영역 선택 가능.

부족한 유형을 선택하여 집중적으로 훈련.

무한 문제 생성 알고리즘 탑재로 중복 없는 학습 가능.

3. ⚔️ 길거리 수학 배틀 (Street Math Battle)

1 vs 1 로컬 멀티플레이 지원.

3라운드 동안 펼쳐지는 킬러 문항 진검승부.

One Chance 룰: 기회는 단 한 번! 오답 시 기회 박탈 및 감점.

긴박감 넘치는 3분 타임어택.

🛠 기술 스택 (Tech Stack)

Core: HTML5, CSS3, JavaScript (ES6+)

Library: React 18 (CDN), Babel (Standalone)

Styling: Tailwind CSS (CDN)

Math Engine: MathJax (수식 렌더링)
